    Documentazione in italiano

    • Appunti di informatica Libera (http://a2.swlibero.it)
      • Monumentale raccolta di istruzioni sulla installazione e configurazione di una macchina Linux (sia server che desktop)
    • Linux Facile (http://www.linuxfacile.it)
      • Manuale per iniziare l'uso di Linux, contiene un (parziale) elenco di comandi utili
    • Italian Linux Documentation Project (http://ildp.pluto.linux.it)
      • Elenco di HOWTO e di guide tradotti in italiano

    Documentazione non in italiano

    • The Linux Documentation Project(www.tldp.org)
      • Raccolta di HOWTO, mini HOWTO e guide per l'uso e la configurazione
    • Linux Gazette (www.linuxgazette.com)
      • Storica rivista elettronica legata a Linux

    Notizie

    • Punto informatico (http://punto-informatico.it)
      • Discreta pubblicazione quotidiana tutta italiana
    • Linux weekly Magazine (www.lwm.net)
      • Settimanale sul mondo Linux
    • Slashdot.org (www.slashdot.org)
      • Tutte le notizie, ma veramente tutte nel campo informatico

    Ricerca di programmi (1)

    • Freshmeat (www.freshmeat.net)
      • L'indicizzatore di software per antonomasia
    • Souceforge (www.souceforge.com)
      • Contiene molti progetti legati a Linux, contiene molti strumenti per interagire con gli sviluppatori
    • Savannah (http://savannah.gnu.org)
      • L'equivalente di souceforge ma dedicato al software GPL, gestito dalla GNU (en)

    Ricerca di programmi (2)

    • SAL (http://sal.kachinatech.com)
      • Archivio di applicazioni scientifiche per Linux
    • Schoolforge (www.schoolforge.net)
      • Come sourceforge, ma dedicato al mondo scolastico
